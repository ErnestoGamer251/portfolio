My Work

tic-tac-toe

Tic-Tac-Toe Game Overview: Tic-Tac-Toe is a two-player game where each player (X or O) takes turns marking a 3x3 grid. The goal is to align three of your symbols in a row, column, or diagonal. If all squares are filled without a winner, the game ends in a draw. Why I Built It: I developed this Tic-Tac-Toe game as a project for The Odin Project bootcamp, where I learned the basics of programming, user interaction, and game design.
